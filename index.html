About

Counting Brain Cells started out as a simple project to learn GUI development. I wanted to learn how to use the PyQt5 GUI framework for Python so I decided to make a simple quiz app. I thought it wouldn't take long but instead will help me understand the basics of GUI development. But then I realised how much effort goes into making an app like this and how much you can and add and do with it. So that's when I renamed it to Counting Brain Cells and developed it into a full usable and enjoyable app.

The app's purpose is to provide a simple and clean interface so you can quiz or revise without distractions. It has an easy to use JSON system so it is very easy to create and share your own quizzes. The app also has direct integration with Open Trivia Database, so you can just pick a category and difficulty and get access to over 4000 questions in the minimilistic Counting Brain Cells interface.

Features

  • Simple interface.

  • Easy to use JSON system to create and share your own quizzes.

  • Open Trivia Database integration allows you to access over 4000 questions for free.

  • At the end of a quiz, you are given the amount of brain cells you have for that topic.
